\begin{equation*} k^{2} - 9 = 0 \end{equation*}

The roots of this equation are:

\begin{equation*} k = 3 \end{equation*}\begin{equation*} k = -3 \end{equation*}

According to the possible solutions of this type of equation, have

\begin{equation*} y = C_{1}e^{\lambda_{1}t} + C_{2}e^{\lambda_{2}t} \end{equation*}

Where $\lambda_{1}$ and $\lambda_{2}$ are the roots of the characteristics eqaution

Rewriting the equation we get,

\begin{equation*} y = C_{1}e^{3t} + C_{2}e^{-3t} \end{equation*}

Equation 1.2 [5 points]¶

Find the general solution to the differential equation \begin{equation*} y'' - 9y = 9t^2 + 4t \end{equation*}

Student's answer Score: 3.0 / 3.0 (Top)

Splitting the solution in the complimentary part and the particular solution.

\begin{equation*} y_t = y_{c} + y_{p} \end{equation*}

Calculating the complimentary part first,

The complimentary equation becomes-

\begin{equation*} k^{2} - 9 = 0 \end{equation*}

the roots of the equation becomes \begin{equation*} k = 3 \end{equation*}

\begin{equation*} k = -3 \end{equation*}

The complimentary solution for the same would become-

\begin{equation*} y_{c} = C_{1}e^{-3t} + C_{2}e^{3t} \end{equation*}

Calculating the particular solution of the equation,

according to the Right Hand Side of the original equation, we can guess the value of our particular solution,

\begin{equation*} y_{p} = At^{2} + Bt + C \end{equation*}

calculating $y_{p}'$ and $y_{p}''$ according to the above equation.

\begin{equation*} y_{p}' = 2At + B \end{equation*}\begin{equation*} y_{p}'' = 2A \end{equation*}

Substituting the above values in the base equation, we get

\begin{equation*} 2A - 9(At^{2} + Bt + C) = 9t^{2} + 4t \end{equation*}

Simplifying the equation,

\begin{equation*} 2A - 9At^{2} - 9Bt - 9C = 9t^{2} + 4t \end{equation*}

equating the terms on both the sides, we get

\begin{equation*} A = -1 \end{equation*}\begin{equation*} B = \frac {-4}{9} \end{equation*}\begin{equation*} C = \frac{-2}{9} \end{equation*}

The particular solution of the equation becomes as-

\begin{equation*} y_{p} = -t^{2} - \frac {4}{9}t -\frac{2}{9} \end{equation*}

Since, we assumed that

\begin{equation*} y_{t} = y_{c} + y_{p} \end{equation*}

combining the equations obtained from above calculations, we get

\begin{equation*} y_{t} = c_{1}e^{3t} + c_{2}e^{-3t} -t^{2} - \frac {4}{9}t -\frac{2}{9} \end{equation*}

Equation 1.5 [5 points]¶

Use variation of parameters to find a solution to the IVP \begin{equation*} y'' - 3y' + 2y = t, \; y(0) = \frac{9}{4}, \; y'(0) = 3 \end{equation*}

Student's answer Score: 3.0 / 3.0 (Top)

Assuming the soultion of the above equation as

\begin{equation*} y_{t} = y_{p} + y_{c} \end{equation*}

where $y_{t}$ is the complete solution, $y_{c}$ is the complimentary solution and $y_{p}$ is the particular solution

CALCULATING THE COMPLIMENTARY SOLUTION COMPONENT-

the characteristic equation of the above differential equation would be-

\begin{equation*} k^{2} - 3k + 2 = 0 \end{equation*}

roots for the characteristic equations are-

\begin{equation*} k = 2 \end{equation*}\begin{equation*} k = 1 \end{equation*}

the corresponding complimentary solution would be-

\begin{equation*} y_{c} = c_{1}e^{t} + c_{2}e^{2t} \end{equation*}

CALCULATION OF PARTICULAR SOLUTION COMPONENT USING VARIATION OF PARAMETERS-

particular solution in variation parameters given as

\begin{equation*} y_p(t) = u_1(t)y_1(t) + u_2(t)y_2(t) \end{equation*}

First, we can calculate the Wronskian as below:

\begin{equation*} W = \left\lvert \begin{matrix} e^{t} & e^{2t} \\ e^{t} & 2e^{2t} \end{matrix} \right\lvert = e^{t}2e^{2t} - e^{t}e^{2t} = e^{3t} \end{equation*}

We can now find $u_1$ and $u_2$:

\begin{align*} u_1 &= -\int{\frac{e^{2t}t}{e^{3t}}dt} = -\int{te^{-t} dt}\\ u_2 &= -\int{\frac{e^{t}t}{e^{3t}}dt} = -\int{te^{-2t} dt}\\ \end{align*}

using the integration by part for u1 & u2:

\begin{align*} u_1 &= -t\int{e^{-t} dt} + \int{t'(\int{e^{-t}dt})dt} = te^{-t} + e^{-t}\\ u_2 &= t\int{e^{-2t} dt} - \int{t'(\int{e^{-2t}dt})dt} = t\frac{e^{-2t}}{-2} - \int{\frac{e^{-2t}}{-2}dt}\\ u_2 &= -e^{-2t}\frac{t}{2} - e^{-2t}\frac{1}{4} = -e^{-2t}(\frac{t}{2} + \frac{1}{4}) \end{align*}

Substituting above value in y_p(t) gives:

\begin{equation*} y_p(t) = e^t(te^{-t} + e^{-t}) - e^{2t}(e^{-2t}(\frac{t}{2} + \frac{1}{4})) \\ y_p(t) = t+ 1 - \frac{t}{2} - \frac{1}{4} = \frac{t}{2} + \frac{3}{4} \end{equation*}

So the particular solution becomes -

\begin{equation*} y_{p} = \frac{1}{2}t + \frac{3}{4} \end{equation*}

Combining both complimentary and particular solution, we get

\begin{equation*} y_{t} = y_{c} + y_{p} \end{equation*}\begin{equation*} y_{t} = c_{1}e^{t} + c_{2}e^{2t} + \frac{1}{2}t + \frac{3}{4} \end{equation*}

Calculating the first derivative of the above function,

\begin{equation*} y_{t}' = c_{1}e^{t} + 2c_{2}e^{2t} + \frac{1}{2} \end{equation*}

substituting the initial values in both the above equations we get

\begin{equation*} y_{t}(0) = c_{1} + c_{2} + \frac{3}{4} = \frac{9}{4} \end{equation*}\begin{equation*} y_{t}(0) = c_{1} + 2c_{2} + \frac{1}{2}= 3 \end{equation*}

Solving above equations we get,

\begin{equation*} c_{1} = \frac{1}{2} \end{equation*}\begin{equation*} c_{2} = 1 \end{equation*}

Plugging in the values and obtaining the final solution, we get

\begin{equation*} y_{t} = \frac{1}{2}e^{t} + e^{2t} + \frac{1}{2}t + \frac{3}{4} \end{equation*}

Introduction¶

The following section is an excerpt from Fundamentals of Differential Equations 8th Ed. by Nagle, Saff, and Snider.

"A mathematical model for the population dynamics of competing species, one a predator with population $x_2(t)$ and the other its prey with population $x_1(t)$ , was developed independently in the early 1900s by A. J. Lotka and V. Volterra. It assumes that there is plenty of food available for the prey to eat, so the birthrate of the prey should follow the Malthusian or exponential law (see Section 3.2); that is, the birthrate of the prey is $Ax_1$ , where $A$ is a positive constant. The death rate of the prey depends on the number of interactions between the predators and the prey. This is modeled by the expression $Bx_1x_2$ , where $B$ is a positive constant. Therefore, the rate of change in the population of the prey per unit time is $\frac{dx_1}{dt} = Ax_1 - Bx_1x_2$ . Assuming that the predators depend entirely on the prey for their food, it is argued that the birthrate of the predators depends on the number of interactions with the prey; that is, the birthrate of predators is $Dx_1x_2$ , where $D$ is a positive constant. The death rate of the predators is assumed to be $Cx_2$ because without food the population would die off at a rate proportional to the population present. Hence, the rate of change in the population of predators per unit time is $\frac{dx_2}{dt} = -Cx_2 + Dx_1x_2$ . Combining these two equations, we obtain the Volterra–Lotka system for the population dynamics of two competing species:

\begin{align*} x_1' = Ax_1 - Bx_1x_2 &\quad\quad(1) \\ x_2' = -Cx_2 + Dx_1x_2 &\quad\quad(2) \end{align*}

Such systems are in general not explicitly solvable."

What is described here is the modelling of population dynamics using a system of non-linear first-order differential equations. The equations of the form shown above are also called Lotka–Volterra equations or predator-prey equations; (1) is the equation for the rate of change of the prey population and (2) is the equation for the rate of change of the predator population.

RK4 for solving systems of equations¶

Euler's method, which we looked at in the previous lab class and assignment, belongs to the family of Runge-Kutta methods for solving ODEs. As we've mentioned in the lab, Euler's method is a first-order method; higher-order Runge-Kutta methods also consider the slopes at points between the current and next step instead of just the slope at the current point and are thus more accurate (but also more computationally demanding).

A commonly used Runge-Kutta method in practice is the fourth-order Runge-Kutta method (aka RK4). Given a first-order ODE, RK4 estimates the value of the function at consecutive steps using the following set of equations:

\begin{gather*} y^\prime = f(t, y) \\~\\ k_1 = h \cdot f(t_n,y_n) \\ k_2 = h \cdot f(t_n + \frac{h}{2}, y_n + \frac{k_1}{2}) \\ k_3 = h \cdot f(t_n + \frac{h}{2}, y_n + \frac{k_2}{2}) \\ k_4 = h \cdot f(t_n + h, y_n + k_3) \\~\\ t_{n+1} = t_n + h \\ y_{n+1} = y_n + \frac{1}{6}[k_1 + 2k_2 + 2k_3 + k_4] \end{gather*}

This is how we can use RK4 for solving a single differential equation, but we now need to extend the set of equations so that we can use the method for solving systems.

such that, given a time step $h$, we know that

\begin{equation*} t_{n+1} = t_n + h \end{equation*}

     return A*x1 - B*x1*x2

     return -C*x2 + D*x1*x2

    x1,x2 = y
    return np.array([f(t,x1,x2), g(t,x1,x2)])

def RK4(t_initial, x1_initial, x2_initial, h, n ):
    
    # source : https://smath.com/wiki/GetFile.aspx?File=Examples/RK4SystemEquations.pdf&AspxAutoDetectCookieSupport=1 
    # intial condition
    t, x1, x2 = t_initial, x1_initial, x2_initial
    result = list()
    y = np.array([x1,x2])
    
    for i in range(n):
        result.append([t,y])
        k1 = h * (func(  t,       y) )
        k2 = h * (func( (t+h/2), (y+k1/2)) )
        k3 = h * (func( (t+h/2), (y+k2/2)) )
        k4 = h * (func( (t+h),   (y+k3)) )
        y =  y + (k1 + 2*k2 + 2*k3 + k4) /6
        t = round(t + h , 2)
    return np.array(result, dtype=object)
